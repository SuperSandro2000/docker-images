PrivateBin Docker Image

## Docker compose

````yaml
---
version: "3"
services:
  privatebin:
    image: supersandro2000/privatebin
    environment:
      - NAME=PrivateBin
      - DB_HOST=db_privatebin
      - DB_NAME=privatebin
      - DB_PASSWORD=secret_password
      - DB_USER=privatebin
    restart: unless-stopped
````
