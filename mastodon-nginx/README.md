Mastodon Nginx Server

## Docker compose

````yaml
version: "2.4"
services:
  db:
    ...
  mastodon:
    ...
  mastodon-sidekiq:
    ...
  mastodon-streaming:
    ...
  redis:
    ...

  mastodon-nginx:
    image: mastodon-nginx
    volumes:
      - ./public/system:/mastodon/public/system
    environment:
      - MASTODON_SERVER=mastodon
      - MASTODON_STREAMING=mastodon-streaming
      - NGINX_DOMAIN=example.com
````
