## Docker compose

````yaml
---
version: "3"
services:
  iodine:
    image: supersandro2000/iodine
    cap_add:
      - NET_ADMIN
    environment:
      - DOMAIN: example.com
      - PASSWORD: CHANGEME!
    restart: unless-stopped
````
